# astrostaat_2026

This repository contains notebooks which go along with the lecture **Statistical methods for astrophysics and cosmology**.

## Structure

- `notebooks/`: chapter notebooks for the lecture.
- `sheets/`: exercise sheets used in class.
	- `sheet_1.ipynb`: exercise version (student tasks).
	- `sheet_1_solution.ipynb`: worked solutions.
	- `data/`: input files used by the sheet exercises (e.g. count-data files for fitting examples).
- `plots_lecture/`: figures and plot assets used in lecture material.
